Gmail Setup Instructions:

  1. Select and copy the HTML between "START GMAIL SIGNATURE" and "END GMAIL SIGNATURE" above
  2. Open Gmail → Settings (gear icon) → "See all settings"
  3. Go to "General" tab → Scroll to "Signature" section
  4. Click the formatting toolbar icon (looks like "Aa") → "Edit HTML"
  5. Paste the copied HTML code
  6. Click "OK" → Save Changes

Note: The gradient background may display differently across email clients, but will work well in Gmail and modern email apps.
